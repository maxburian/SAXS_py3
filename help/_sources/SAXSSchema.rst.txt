.. _required:

 The ':red:`*`' signifies a required Field.

The SAXS configuration file specifies the parameters of a SAXS sensor calibration. It is written in the JSON format which governs the general syntax.


:Type:
  object
:Contains:
  :ref:`Title <Title>`, :ref:`Geometry <Geometry>`:red:`*`, :ref:`Masks <Masks>`:red:`*`, :ref:`Slices <Slices>`, :ref:`Wavelength <Wavelength>`:red:`*`, :ref:`PolarizationCorrection <PolarizationCorrection>`, :ref:`Directory <Directory>`:red:`*`, :ref:`Threads <Threads>`, :ref:`Live-Filelisting <Live-Filelisting>`, :ref:`OverwriteFiles <OverwriteFiles>`, :ref:`GISAXSmode <GISAXSmode>`
:Required:
  True
:JSON Path:
  * :ref:`# <root>` 

Example JSON: 

.. code:: json

    {
      "Title": "Example Calibration",
      "Geometry": {
        "Tilt": {
          "TiltRotDeg": 0,
          "TiltAngleDeg": 0
        },
        "Imagesize": [
          1000,
          900
        ],
        "BeamCenter": [
          800.0,
          400.0
        ],
        "PixelSizeMicroM": [
          100.0
        ],
        "DedectorDistanceMM": 1000.0
      },
      "Wavelength": 1.54,
      "Directory": [],
      "Live-Filelisting": true,
      "OverwriteFiles": true,
      "Threads": 2,
      "Masks": [],
      "Slices": [],
      "GISAXSmode": false
    }

.. _Tilt:

Tilt
-------------------------

The sensor, usually is not perfectly perpenticular to the ray direction. The tilt angle can be specified by giving the following paramters.


:Type:
  object
:Contains:
  :ref:`TiltRotDeg <TiltRotDeg>`:red:`*`, :ref:`TiltAngleDeg <TiltAngleDeg>`:red:`*`
:Required:
  True
:JSON Path:
  * :ref:`# <root>` [':ref:`Geometry <Geometry>`'][':ref:`Tilt <Tilt>`']

Example JSON: 

.. code:: json

    {"Tilt": {"TiltRotDeg": 0,"TiltAngleDeg": 0}}

.. _Oversampling:

Oversampling
-------------------------

Oversampling factor for radial integration. The higher, the longer the setup but the higher the accuracy. More then 3 is probably overkill. 


:Type:
  integer
:Required:
  True
:Default:
  3
:JSON Path:
  * :ref:`# <root>` [':ref:`Masks <Masks>`'][0][':ref:`Oversampling <Oversampling>`']

Example JSON: 

.. code:: json

    {"Oversampling": 3}
