=============================
The Dedector Calibration File
=============================

The :py:class:`SAXS.calibration` class and the saxsdog tool accepts a input file with the calibration data. 
This input file is written as a JSON file. This is a common syntax to express structured data as text. 
You might want to read a bit about it before moving on.
